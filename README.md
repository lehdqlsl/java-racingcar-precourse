# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 요구사항분석
1. 경주할 자동차 이름 입력 받기.
   1. 5자 이하 확인 
   2. 자동차 클래스
2. 시도할 회수 입력 받기
3. 0~9 사이 랜덤값 생성
   1. 0~9값 검증
   2. 4이상: 전진
   3. 3이하: 멈춤(아무것도하지않음)
   4. 화면 출력
   5. 입력받은 회수만큼 반복
4. 게임 결과 확인
   1. 가장 멀리 이동한 자동차 추출
